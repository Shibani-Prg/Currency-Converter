# Currency Converter

A simple web application that allows users to convert currencies from one country to another using real-time exchange rates.

## Features

* Convert between multiple currencies
* Fetch real-time exchange rates using API
* User-friendly interface
* Responsive design for mobile and desktop


## Usage

1. Select the currency you want to convert from.
2. Select the currency you want to convert to.
3. Enter the amount.
4. Click **Convert** to see the result.

## Technologies Used

* HTML
* CSS
* JavaScript
* Exchange Rates API

## API Used

This project uses the Exchange Rates API to fetch real-time currency conversion rates.
